# Инструкции по сборке исполняемого файла

## Описание

В проект Strod-Service Technology добавлены скрипты для автоматической компиляции приложения в исполняемый файл, который можно запустить без установленного Python.

## Содержимое

- `build_exe.py` - основной скрипт для компиляции приложения в исполняемый файл
- `setup_build.py` - скрипт для установки зависимостей и запуска компиляции
- `compile_to_exe.bat` - Windows-скрипт для запуска компиляции
- `compile_to_exe.sh` - Linux/Mac-скрипт для запуска компиляции
- `create_exe.bat` - упрощенный Windows-скрипт для компиляции
- `create_exe.sh` - упрощенный Linux/Mac-скрипт для компиляции
- `README_BUILD.md` - подробное описание процесса компиляции

## Быстрая сборка

### Вариант 1: Использование готовых скриптов

#### Для Windows:
- Запустите `create_exe.bat` или `compile_to_exe.bat`

#### Для Linux/Mac:
- Запустите `./create_exe.sh` или `./compile_to_exe.sh`

### Вариант 2: Использование Python

1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите сборку: `python build_exe.py`

### Вариант 3: Автоматическая установка и сборка

1. Запустите: `python setup_build.py`

## Результат

После успешной компиляции будет создан исполняемый файл:
- В Windows: `StrodService.exe`
- В Linux/Mac: `StrodService`

Этот файл содержит все необходимые зависимости и может работать автономно.

## Замечания

- Компиляция может занять несколько минут
- Для компиляции требуется установленный Python и зависимости из requirements.txt
- Созданный исполняемый файл включает в себя все необходимые ресурсы: папки assets, data, docs, filestorage
