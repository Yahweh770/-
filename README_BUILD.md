# Компиляция проекта Strod-Service Technology в исполняемый файл

## Описание

Этот проект включает в себя скрипты для автоматической компиляции приложения Strod-Service Technology в исполняемый файл (exe для Windows или исполняемый файл для Linux/Mac).

## Файлы

- `build_exe.py` - основной скрипт для компиляции проекта в исполняемый файл
- `compile_to_exe.bat` - скрипт для Windows для запуска компиляции
- `compile_to_exe.sh` - скрипт для Linux/Mac для запуска компиляции
- `StrodService` (или `StrodService.exe`) - скомпилированный исполняемый файл приложения

## Как использовать

### Вариант 1: Использование готовых скриптов

#### Для Windows:
1. Запустите `compile_to_exe.bat`
2. После завершения компиляции запустите `StrodService.exe`

#### Для Linux/Mac:
1. Запустите `./compile_to_exe.sh`
2. После завершения компиляции запустите `./StrodService`

### Вариант 2: Использование Python напрямую

1. Убедитесь, что у вас установлены все зависимости из `requirements.txt`
2. Запустите `python build_exe.py`
3. После завершения компиляции запустите созданный исполняемый файл

## Зависимости

Для компиляции используются следующие зависимости:
- PyInstaller (устанавливается автоматически)
- Все зависимости из `requirements.txt`

## Особенности

- Созданный исполняемый файл включает в себя все необходимые библиотеки и ресурсы
- Приложение может работать автономно без установленного Python
- Включены все необходимые папки: assets, data, docs, filestorage
- Поддерживается автономная работа с базой данных SQLite

## Запуск скомпилированного приложения

После компиляции создается один исполняемый файл, который можно запустить как обычное приложение. При первом запуске приложение создаст необходимые директории и файлы в папке, где находится исполняемый файл.
