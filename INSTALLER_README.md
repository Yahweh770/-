# Установщик приложения

В этом репозитории представлены два варианта установщика приложения:

## 1. Графический установщик (GUI)

Файл: `installer_demo.py`

Графический установщик с использованием библиотеки `tkinter`, который предоставляет полноценный интерфейс с:
- Согласием с лицензионным соглашением
- Выбором пути установки
- Визуализацией процесса установки
- Финальными опциями (открыть README, запустить приложение)

### Особенности:
- Интерактивный графический интерфейс
- Прогресс-бар установки
- Лог установки в реальном времени
- Поддержка выбора пути установки
- Опции после завершения установки

## 2. Консольный установщик

Файл: `console_installer.py`

Консольный установщик, который работает в терминале и демонстрирует ту же логику установки:
- Согласие с лицензией
- Выбор пути установки
- Имитация процесса установки с визуализацией прогресса
- Создание файлов приложения
- Опции после установки

### Особенности:
- Работает в консоли без графического интерфейса
- Интерактивное взаимодействие с пользователем
- Создание фиктивных файлов приложения
- Поддержка дополнительных действий после установки

## Запуск установщиков

### Графический установщик:
```bash
python installer_demo.py
```

> **Примечание:** Для запуска графического установщика необходима система с поддержкой графического интерфейса и установленной библиотекой Tk.

### Консольный установщик:
```bash
python console_installer.py
```

> **Примечание:** Консольный установщик работает в любой системе с Python.

## Реальные установщики

Для создания настоящих установщиков в формате .exe с полнофункциональным графическим интерфейсом рекомендуется использовать специализированные инструменты:

- **NSIS (Nullsoft Scriptable Install System)** - мощный скриптовый установщик
- **Inno Setup** - популярный инструмент для создания установщиков
- **WiX Toolset** - инструмент для создания установщиков с использованием XML

Эти инструменты позволяют создавать полноценные .exe установщики с:
- Графическим интерфейсом
- Согласием с лицензией
- Выбором компонентов
- Настройкой параметров установки
- Созданием ярлыков
- Интеграцией с системой (регистрация в "Удалении программ")
- Подписью установщика
