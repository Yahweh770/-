# Руководство по использованию установщиков

## Введение

В этом руководстве описаны способы использования созданных установщиков и их особенности.

## Установщики

### 1. Графический установщик (`installer_demo.py`)

#### Описание
Графический установщик с полноценным интерфейсом, созданный с использованием библиотеки tkinter.

#### Запуск
```bash
python installer_demo.py
```

> **Важно**: Для запуска требуется система с графическим интерфейсом и установленной библиотекой Tkinter.

#### Функции
1. **Согласие с лицензией**:
   - Полноэкранное окно с текстом лицензии
   - Возможность прокрутки текста
   - Чекбокс для подтверждения согласия

2. **Выбор пути установки**:
   - Интерактивное окно выбора папки
   - Возможность ввода пути вручную
   - Проверка доступности папки

3. **Процесс установки**:
   - Прогресс-бар с процентным индикатором
   - Текстовое поле с логом установки
   - Многопоточная обработка (не блокирует интерфейс)

4. **После установки**:
   - Возможность открыть файл README
   - Возможность запустить установленное приложение

### 2. Консольный установщик (`console_installer.py`)

#### Описание
Консольный установщик, работающий в терминале, с интерактивным процессом установки.

#### Запуск
```bash
python console_installer.py
```

#### Функции
1. **Согласие с лицензией**:
   - Отображение текста лицензии в консоли
   - Подтверждение через ввод 'y'/'yes'/'да' или 'n'/'no'/'нет'

2. **Выбор пути установки**:
   - Ввод пути через командную строку
   - Возможность использования пути по умолчанию
   - Проверка и создание папки при необходимости

3. **Процесс установки**:
   - Процентный индикатор прогресса
   - Имитация различных этапов установки
   - Создание фиктивных файлов приложения

4. **После установки**:
   - Возможность просмотра README
   - Имитация запуска приложения
   - Выбор дополнительных действий

## Утилиты

### 1. Скрипт запуска (`run_installers.sh`)

#### Описание
Bash-скрипт для удобного выбора и запуска установщиков.

#### Использование
```bash
./run_installers.sh
```

### 2. Тестовый скрипт (`test_installers.py`)

#### Описание
Скрипт для проверки работоспособности установщиков без интерактивного ввода.

#### Использование
```bash
python test_installers.py
```

## Сравнение установщиков

| Характеристика | Графический | Консольный |
|----------------|-------------|------------|
| Требует GUI | Да | Нет |
| Визуализация | Прогресс-бар + лог | Процентный индикатор |
| Удобство | Высокое | Среднее |
| Платформа | Кроссплатформенный | Кроссплатформенный |
| Зависимости | Python + tkinter | Только Python |
| Пользователь | Конечный | Технически продвинутый |

## Создание настоящего установщика

Для создания настоящего установщика в формате .exe рекомендуется использовать:

### NSIS
- Файл примера: `nsis_example.nsi`
- Язык сценариев для создания установщиков
- Подходит для большинства приложений

### Inno Setup
- Pascal-подобный синтаксис
- Отличная поддержка Windows
- Хорошая документация

### WiX Toolset
- XML-базированный подход
- Интеграция с MSI
- Подходит для enterprise решений

## Заключение

Созданные установщики демонстрируют все ключевые функции, которые должны присутствовать в полноценном установщике:

1. ✅ Согласие с лицензионным соглашением
2. ✅ Выбор пути установки
3. ✅ Визуализация процесса установки
4. ✅ Действия после установки
5. ✅ Интерактивный интерфейс

Они могут служить основой или шаблоном для разработки более сложных решений с использованием специализированных инструментов.
