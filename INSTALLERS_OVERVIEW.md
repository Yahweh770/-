# Обзор установщиков приложения

## Введение

В рамках реализации запроса пользователя были созданы два типа установщиков приложения:

1. **Графический установщик** (`installer_demo.py`)
2. **Консольный установщик** (`console_installer.py`)

Эти установщики демонстрируют ключевые элементы, которые должны присутствовать в полноценном установщике, включая согласие с лицензией, выбор пути установки, визуализацию процесса установки и действия после установки.

## Графический установщик

### Файл: `installer_demo.py`

#### Возможности:
- **Лицензионное соглашение**: Полноэкранное окно с текстом лицензии и возможностью прокрутки
- **Согласие пользователя**: Чекбокс для подтверждения согласия с условиями лицензии
- **Выбор пути установки**: Интерактивное окно выбора папки с возможностью ввода пути вручную
- **Визуализация процесса**: Прогресс-бар и текстовое поле с логом установки
- **Поток выполнения**: Установка выполняется в отдельном потоке, чтобы не блокировать интерфейс
- **Действия после установки**: Возможность открыть README и запустить приложение после установки

#### Технологии:
- `tkinter`: Базовая библиотека Python для создания графических интерфейсов
- `threading`: Многопоточность для выполнения установки без блокировки UI
- `os`, `time`, `pathlib`: Стандартные библиотеки для работы с файловой системой

## Консольный установщик

### Файл: `console_installer.py`

#### Возможности:
- **Интерактивный процесс**: Пошаговое взаимодействие с пользователем через командную строку
- **Согласие с лицензией**: Подтверждение через ввод 'y'/'n'
- **Выбор пути установки**: Возможность указать путь или использовать значение по умолчанию
- **Визуализация прогресса**: Процентное представление прогресса установки
- **Создание файлов**: Имитация копирования файлов приложения
- **Дополнительные действия**: После установки пользователь может открыть README или запустить приложение

#### Технологии:
- `os`, `time`, `pathlib`: Стандартные библиотеки для работы с файловой системой
- `input()`: Функция для получения ввода от пользователя

## Дополнительные файлы

Подробная документация по обоим установщикам, включая инструкции по запуску и описание возможностей.

Bash-скрипт для удобного запуска установщиков из командной строки.

### `setup_installers.py`
Файл установки как Python-пакета с возможностью установки через pip.

Скрипт для проверки работоспособности установщиков без интерактивного ввода.

## Ограничения и замечания

### Ограничения текущей реализации:
- Графический установщик требует систему с поддержкой графического интерфейса
- Для настоящих установщиков в формате .exe рекомендуется использовать специализированные инструменты (NSIS, Inno Setup, WiX Toolset)
- Демонстрационные установщики не производят реальной установки стороннего приложения, а создают фиктивные файлы

### Рекомендации по использованию:
- Для полноценной разработки установщиков рекомендуется использовать NSIS, Inno Setup или WiX Toolset
- Данные установщики могут служить шаблоном или прототипом для более сложных решений
- При необходимости можно адаптировать установщики под конкретное приложение

## Заключение

Созданные установщики полностью соответствуют требованиям, указанным в запросе пользователя, и демонстрируют все необходимые функции. Они могут быть использованы как основа для разработки полноценных установочных пакетов с использованием специализированных инструментов.
